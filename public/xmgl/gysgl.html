# 供应商管理

# 1. 往来单位

说明：往来单位以统一社会信用代码为唯一识别码，禁止重复。在选择之前，请查找是否有这家单位，如没有，则新增；如有，则可以直接选择使用。

# 2. 往来分类

说明：通过对供应商的分类，企业可以针对不同类别的供应商制定个性化的业务流程和管理规范。
