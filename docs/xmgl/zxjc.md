# 10.专项检查

## 10.1 专项检查

![专项检查](./imgs/zxjc/zxjc.png)

### 功能说明
专项检查是针对项目特定方面进行的深度检查，确保符合：
- 行业标准
- 管理流程  
- 项目要求

### 检查特点
1. **针对性**：聚焦特定领域/问题/风险
2. **专业性**：由管理部门或专业机构实施
3. **目的性**：
   - 发现问题隐患
   - 评估风险等级
   - 确保合规性
   - 推动整改提升

### 填写要求
*红色标注字段为必填项*
